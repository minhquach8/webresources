Kiểu dữ liệu (Datatype)

  • Kiểu dữ liệu nguyên thủy (Primitive Types):
    • string: kiểu chuỗi, được đặt trong cặp dấu nháy đổi (“ “) hoặc nháy đơn (‘ ‘)
    • number: số thập phân, số nguyên
    • boolean: là dữ liệu chỉ có 2 giá trị True hoặc False
      • Có các giá trị sẽ luôn trả về là False, chúng được gọi là Falsy Values:
        • 0
        • false
        • ‘’ hoặc “” (không có khoảng trắng)
        • undefined
        • NaN
        • null
    • undefined: kiểu dữ liệu chưa xác định
    • null: biến không có giá trị
    • BigInt: dùng cho những số nguyên cực lớn hơn 2^53 -1 và nhỏ hơn -(2^53-1)
    • Symbol: tạo ra các giá trị duy nhất (unique value) và bất biến (immutable)
  • Kiểu dữ liệu phức tạp (Complex Data):
    • Function
    • Objects: kiểu dữ liệu tham chiếu, gồm các cặp key - value. Ví dụ:
    let myObj = {
	a : 1,
	b : 'hello',
	10 : [1, 'cat', 4],
	myFunction: function() {
		//TODO
	}
}

❗Phân biệt cơ bản null và undefined:

  • Khi khai báo một biến, nhưng ta không gán giá trị => undefined
  • Null được cấp phát cho biến, và nó cho biết rằng biến đã được định nghĩa (defined) nhưng thiếu giá trị (missing value) theo chủ đích
    • let test = null;

💡Cách kiểm tra kiểu dữ liệu:

console.log(typeof 3) //number

Toán tử số học - Arithmetic

❗Toán tử ++ và -- sẽ có hai trường hợp sau:

  • Prefix - Tiền tố ++x:
    • let x = 5;
    • (++x) sẽ trả kết quả 6; có nghĩa là x + 1 = 5 + 1 = 6 và gán vào biến x, rồi mới trả về.
  • Postfix - Hậu tố x++:
    • let x = 5;
    • (x++) sẽ trả kết quả 5; có nghĩa là x + 1= 5 + 1 = 6 nhưng không trả về ngay mà lần kế gọi x, khi đó ta mới có x = 6.

📖Bài tập thử:

let number = 6;
let output = number++ + --number; // Result = 12

let number2 = 6;
let output2 = ++number2 * 2 - number2-- * 2; // Result = 0

💡Thường đi kèm với lệnh điều kiện

💡Giá trị trả về là True hoặc False

❗Lưu ý: JavaScript có hai dạng so sánh:

  • So sánh trừu tượng (abstract equality): ==
  • So sánh nghiêm ngặt (strict equality): ===

📑Tương tự cho khác nhau !== và !=

💡Sự khác biệt:Toán tử so sánh trừu tượng sẽ cố gắng giải quyết các kiểu dữ liệu thông qua việc chuyển đổi kiểu dữ liệu trước khi so sánh, còn toán tử so sánh nghiêm ngặt sẽ trả về false nếu các giá trị khác nhau.

⇒ Nên sử dụng so sánh nghiêm ngặt, vì sẽ làm tăng sự rõ ràng của code và giúp ngăn chặn những sai sót phát sinh từ việc sử dụng so sánh trừu tượng.

Toán tử logic - Logic

💡Nâng cao:

let result = 'A' && 'B' && 'C';
console.log(result); // C

let result = 'A' && null && 'C';
console.log(result); // null
  • Khi giá trị khác các giá trị thuộc Falsy Values, điều kiện && sẽ tiếp tục thực hiện qua phải đến hết và trả giá trị cuối. Nếu gặp phải giá trị Falsy Value sẽ dừng lại ngay và trả về đúng giá trị đó.
let result = 'A' || 'B' || 'C';
console.log(result); // A

let result = 'A' || undefined || 'C';
console.log(result); // A
  • Trường hợp || thì cứ gặp đúng giá trị khác Falsy Values thì trả về giá trị đó luôn.

Mảng - Array

  • Mảng được dùng để chứa nhiều giá trị (bất kỳ Datatypes nào) và có thể truy cập bằng chỉ số vị trí
  • Kiểm tra có phải array hay không?
    Array.isArray(x);
  • Thuộc tính và phương thức:
    const myArr = [1, 3, 'b', 2, 'e', 5];

// Access a value
myArr[2];

// Converts an array to a string of (comma separated) array values
myArr.toString();

// Joins all array elements into a string
myArr.join('-');

// Removes the last element from an array
myArr.pop();

// Add new item at the end of an array
myArr.push('f');

// Removes the first array element and "shifts" all other elements to a lower index.
myArr.shift();

// Adds a new element to an array (at the beginning), and "unshifts" older elements:
myArr.unshift('g');

// Creates a new array by merging (concatenating) existing arrays
const arr2 = [8, 9, 10];
const newArr = myArr.concat(arr2);

// Splice() 
	// The first parameter (2) defines the position where new elements should be added
	// The second parameter (0) defines how many elements should be removed.
	// The third parameter defines what element should be added
myArr.splice(2, 0, 'hello');

// Slices out a piece of an array into a new array.
myArr.slice(3, 6);

Hàm - Function

  • Hàm là một khối mã và làm 1 việc cụ thể
  • Một số hàm Built-in
    • Đây là các hàm được xây dựng sẵn trong JS, có thể gọi và sử dụng liền
    • Một số hàm cơ bản và thường dùng:
      alert();
console.log();
confirm();
prompt();
setTimeout();
setInterval();
//setInterval() khác chỗ sẽ chạy sau cacác khoảng thời gian bằng nhau
  • Hàm không thực thi khi được định nghĩa, sẽ được thực hiện khi được gọi
  • Hàm có thể nhận tham số
  • Hàm có thể trả về 1 giá trị (mặc định là undefined) và khi return sẽ thoát ra khỏi hàm.
  • Tham số (bất kỳ Datatypes nào) là giá trị truyền vào hàm để thực hiện một việc gì đó
function tenHam(param1, param2) {
	//TODO
	return //sth;
}
  • Biến trong hàm sẽ là private, nghĩa là không truy cập được khi ra khỏi hàm (Function Scope). TUY VẬY, JS đã sửa Function Scope thành Block Scope để tránh các lỗi không mong muốn và giống với tư duy của các ngôn ngữ lập trình khác.

💡Các loại function:

  • Declaration function:
    function tenHam(param1, param2) {
	//TODO
	return //sth;
}
  • Expression function
    const tenBien = function (param1, param2) {
	//TODO
}
  • Arrow function
    const tenBien = (param1, param2) => {
	//TODO
}

    Hoặc nếu nội dung hàm chỉ có 1 dòng thì có thể viết gọn hơn nữa:

    const tenBien = (param1, param2) => //TODO;

Vòng lặp - Loop

  • Tránh những đoạn code lặp đi lặp lại
  • Có 2 loại thường dùng là For và While.
    • For sẽ dùng khi biết chính xác số lần lặp

For

  • Ví dụ:
for (let i = 0; i < 100; i++) {
	console.log(i);
}
  • Duyệt mảng:
const myArr = ['JS', 'Python' 'C++'];

for (let i = 0; i < myArr.length; i++) {
	console.log(myArr[i]);
}

💡For - in sẽ duyệt qua key | For - of sẽ duyệt qua value

// For - in
const myInfo = {
	name: John,
	age: 25,
	address: 'New York',
	getName: function() {
		return this.name;
	}
};

for (let key in myInfo) {
	console.log(key);
}

// For - of
const myArr = ['JS', 'Python' 'C++'];
for (let value of myArr) {
	console.log(value);
}

for (let value of Object.values(myInfo)) {
	console.log(value);
}

While

let i  = 0;
while ( i < 100) {
	console.log(i);
	i++;

Do-while

let i = 0;
do {
	i++;
	console.log(i);
} while(i<10);
  • Do-while khác While ở chỗ nó sẽ thực hiện nội dung vòng lặp trước khi kiểm tra điều kiện.
